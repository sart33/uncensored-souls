1. General Information

⚠️ 18+ — When you launch the app for the first time, you will see an age restriction screen. If you are under 18 — press "No".

2. API Keys Setup

Go to: Settings → API Keys

You need to enter two keys:

DeepSeek API (required for chat)

DeepSeek API Keys platform.deepseek.com/api_keys

Minimum deposit: $2. If you set a response limit, this can last for a fairly long time.

Novita AI (for image generation)

Novita AI API Keys novita.ai/dashboard/api-keys

Minimum deposit: $10. One image costs about half a cent, so this is enough for a large number of generations.

  • DeepSeek key entered → chat becomes available
  • Novita key entered → image generation is unlocked
  • API keys are stored only on your device

3. Quick Setup

Chat Text Size

After entering API keys, go to settings and choose a comfortable chat font size.

4. Characters

4.1 Default Characters

Go back to the main screen and open the character panel. You will see two pre-generated characters — Anya and Natasha. They differ in appearance and behavior.

  • 👁 Eye icon → open character profile
  • 📷 Camera icon → generate an image
  • Select category: Romance / Erotic / 18+
  • The last selected category is remembered
  • When opening the gallery from chat, the same category will be used

The profile contains: character description and greeting message.

4.2 Editing and Creating Characters

Press Edit to modify description, appearance, and behavior.

It is recommended to define a consistent character with personality and motivation — this improves response quality and stability.

4.3 Appearance Description Guidelines

You can write the character description in any language.

However, the following attributes are recommended to be written in English:

  • Ethnicity — Caucasian, Asian, Latina, Black, Middle Eastern, etc.
  • Hair color — blonde, brunette, redhead, black hair, etc.
  • Skin tone — fair skin, tan skin, dark skin, olive skin, etc.

This reduces unexpected results during image generation, especially for avatars. If you plan to generate nude images, it is recommended to explicitly describe body features as well.

4.4 Saving and Deleting

After editing, press Save.

⏱ For about 10 seconds, prompts and generation styles are being updated. Avoid generating images during this time.

On mobile, you can also crop and adjust the avatar.

Delete Character removes: the character, all images, and all chat branches. A confirmation dialog will appear before deletion.

4.5 Creating a New Character

  • Enter a name
  • Add a description (you can use default examples)
  • Define behavior (default can also be used as a base)

4.6 Avatar Generation

Click the "Generate" button on the character edit screen. A modal window will appear — centered on desktop, bottom sheet on mobile.

Choose an avatar style:

For all styles except Art Nude: first a prompt is generated, then the image itself.

  • Mobile: tap "Crop and save" (grid icon) to adjust scale and framing. Regeneration is available in the same or different style.
  • Desktop: regenerate in the selected style or click "Generate" again to choose a different one.
  • You can also select any image from your device gallery.

5. Multi-Chat

In the Multi-chat section you will see two default characters — Natasha and Anya. You can start chatting immediately. Each character responds independently.

  • Tap character icon → open their gallery
  • Use Edit to add characters, change behavior, edit greeting
  • Optimal: 2–4 characters · Maximum: up to 5
  • More may result in overly long responses or confusion

6. Chat Usage

Click a character to open their chat list. If no chat exists — create a new one.

6.1 General Flow

After sending a message, response time is 2–30 seconds. Multi-chat may take longer.

6.2 Controlling Responses

You can specify desired response length.

  • Continue → continues the scene
  • More Details → enables detailed mode (actions, emotions, reactions of all participants)

6.3 Image Generation

In single-character chats, there is a Photo button. It generates an image based on the current scene, pose, clothing, and mood. You can regenerate the result.

6.4 Message Editing

Long press on a message to edit, delete, or copy. After editing or deleting, the conversation continues from that point.

6.5 Voice Input (Android)

Voice input via keyboard is recommended. The model generally understands context well, but it's better to review the text before sending.

6.6 Image Storage

🔒 All generated images are stored inside the app and do not appear in the device gallery.

7. Advanced Settings

Creativity (temperature)

  • ~0.7 → stable character behavior
  • 1.2–1.5 → more creative, but possible personality drift

Personality Reminder

Recommended to enable. Optimal frequency: about 1 in 10 messages.

Long Chat Summarization

Allows the character to retain memory in long conversations. Recommended if you plan long-term interaction. The longer the interaction, the higher the limit can be set.

⚠️ Clear All Summaries affects all characters.

Auto-delete Chat Images

Applies only to images generated inside chats. Does not affect character gallery. Range: 7 to 60 days.

AI Response Limit

Controls the maximum length of AI responses (in tokens).

RangeDescription
100–200Minimal cost, short but expressive replies
300–800Balanced mode for regular roleplay
1000–2000Long, detailed responses

Higher values significantly increase cost and are usually unnecessary. In multi-character chats, total usage scales with the number of characters.

User Input Limit

Can be left unchanged — not critical for typical usage.
